Renders a Lucide stroke icon by name — SponsorMatch's icon system. The page must load the Lucide UMD script.

```jsx
<Icon name="shield-check" size={20} />
<Icon name="arrow-right" />
<Icon name="star" color="var(--rating-star)" />
```

Common names in this system: `shield-check` (verified/secure), `star`, `search`, `message-circle`, `bar-chart-3`, `users`, `arrow-right`, `check`, `lock`.
