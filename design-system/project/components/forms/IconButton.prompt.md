Icon-only square button for toolbars, card actions and compact controls. Always pass `label` for accessibility.

```jsx
<IconButton name="heart" label="Merken" />
<IconButton name="more-horizontal" variant="outline" label="Mehr" />
<IconButton name="send" variant="accent" label="Senden" />
```

Variants: `ghost` (default), `outline`, `solid`, `accent`. Sizes: `sm` `md` `lg`.
