Radio buttons grouped by `RadioGroup`. Use for single-choice options like the onboarding role picker.

```jsx
<RadioGroup defaultValue="creator">
  <Radio value="sponsor" label="Ich bin Sponsor" description="Firma, Marke, Marketing-Team" />
  <Radio value="creator" label="Ich werde gesponsert" description="Sportler, Verein, Creator" />
</RadioGroup>
```
