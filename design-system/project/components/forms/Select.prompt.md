Dropdown select matching the Input style. Pass `options` as strings or `{value,label}`.

```jsx
<Select label="Kategorie" placeholder="Wählen…" options={['Fußball','E-Sport','Fitness','Lifestyle']} />
<Select label="Reichweite" options={[{value:'10k',label:'10k+'},{value:'100k',label:'100k+'}]} />
```
